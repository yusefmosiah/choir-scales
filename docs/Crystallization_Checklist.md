## Process

For each file:

1. **Analysis**
   ```
   SEQUENCE analyze_file:
     identify_patterns
     map_to_theory
     list_invariants
     note_dependencies
   ```

2. **Crystallization**
   ```
   SEQUENCE crystallize_file:
     extract_core_logic
     apply_patterns
     add_verification
     document_mapping
   ```

3. **Verification**
   ```
   SEQUENCE verify_file:
     check_invariants
     test_patterns
     validate_mapping
     review_documentation
   ```

## Completion Criteria

File is crystallized when:
- Core patterns are explicit
- Theory mapping is clear
- Error handling is complete
- Tests verify invariants
- Documentation bridges theory and practice
